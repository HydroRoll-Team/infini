title: Module 模块

!!! info
    此对象是 Python 代码的一种组织单位。各模块具有独立的命名空间，可包含任意 Python 对象。模块可通过 [importing] 操作被加载到 Python 中。

    另见 [package]。

    [importing]: https://docs.python.org/zh-cn/3/glossary.html#term-importing
    [package]: ../Package
